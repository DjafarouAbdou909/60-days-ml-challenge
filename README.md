# 60-Days ML Challenge

> 12 datasets · 60 days · progressive difficulty — from classification to advanced ML.

Built to sharpen real-world data science skills, compete on African-focused platforms like Zindi, and document a structured ML learning journey.

---

## ⚡ Core Challenge

> **Every single line of code in this repository is written by hand — no AI-generated code, ever.**

This is the main rule of the challenge. AI tools may be used to understand concepts, review logic, or discuss approaches — but the implementation is 100% mine. This is how real skills are built.
